# Exercicio-blog

Desenvolvido um protótipo de blog e aprimorando os conhecimentos em ReactJS como: 

* JSX & Webpack/Babel
* Create React App
* Props
* JSON Server & API
* Testando a API com Postman
* React Hooks
* Axios
* Context API

## Tecnologias 

Aqui estão as tecnologias que utilizei no projeto:

* HTML
* CSS
* SCSS
* ReactJS
* Postman
* JSON Server
